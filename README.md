### ⚡ Quick Installation

For a quick preview of Task Tacker, get started in seconds:

#### Prerequisites

- Node.js (version 18-22) and npm

#### Steps

```bash
git clone https://github.com/maxon-code/task-tracker
cd task-tracker
npm install
npm run dev
```
